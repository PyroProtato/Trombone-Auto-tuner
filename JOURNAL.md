I did a lot of building work since the last devlog! I will start to list the things I had to do here:
 - Print all of the cadded parts (around 10-12 hours of printing)
 - Use my pinecil soldering iron to heatset ~32 M3 Inserts
 - Place and rewire the electrical components using standoffs
 - Solder the appropriate connections (see previous devlog)
 - Bolt everything in place (and strip a few bolt heads in the process oops...)
 - Disassemble and Reassemble multiple times since this took a few days and I needed to practice trombone like regular without a bunch of cables stabbing me in the neck
 - Adjust things like cable management
 - Troubleshoot issues with motor power (see below)

The thing that ended up taking the most time was with a motor power issue. Both of the linear actuators I am using are recommended for 9V of electricity. When I was testing just the code by itself in the first few devlogs, I was able to use a 5V supply to power one just fine. However, now I realized that when 2 were connected there was not enough voltage to move both, and as such neither moved. This ended up taking a decent amount of time first to diagnose the problem (I thought it was a code issue or an electrical issue because I'm not that familiar with the l298N module) and then to figure out a solution. I ended up taking a 9V battery (that I luckily already made space for in the print) and e-taping wires to both poles to run its 9V into the motor controller. Now it works much better and more powerful.

With all of this settled, the last thing to do is to get all of the finishing touches done and finally test the completed product. Fingers crossed that not _too_ much will need to be changed...

Hey! This journal was mostly a ton of research, purchasing, testing and other things. 

First of all, I had to research ways to make this project possible I knew I wanted to use linear motion in some way to push the tuning slide but that was about it. Using research and a digikey credit card I got from SoM, I found the DFRobot linear actuators were cheap and perfect for this task. I also found that I need a motor controller which I slavaged from an arduino-based car my brother made along with an arduino and some other miscellaneous components.

After my linear actuators came in, I started experimenting with them. It took an embarassingly long time to figure out how to operate the motor controller since it wasn't the standard l298n motor controller. In particular, I had to find out how to wire a 6v power supply separately to the motor controller, which I ended up doing using the battery holders from my snap circuits box (lol). Also, since the linear actuators didn't fit perfectly into the motor board, I had to strip some wires and I will add some solder later as well to stiffen them in order to adapt it over.
